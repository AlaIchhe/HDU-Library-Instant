# HDU 图书馆即时预约

在浏览器中登录一次图书馆系统后，工具会自动读取 cookie、每天定时提交预约。全程在网页中操作，无需手动编辑配置文件。

> 配置文件 `config.yaml` 首次运行时会自动生成，无需手动创建。所有参数在网页中修改并保存即可。

## 初次配置（仅一次）

1. 在浏览器中打开 https://hdu.huitu.zhishulib.com ，登录你的图书馆账号（Chrome 或 Edge）
2. 双击 `start_web.bat` 启动网页控制台，浏览器访问 `http://127.0.0.1:8765`
3. 点击状态栏的 **获取cookie** 按钮，等待显示 "Cookie: ✅ 浏览器自动读取"
4. 在左侧表单中选择楼层、填写座位号、选开始时间和时长，点 **保存计划**
5. 勾选 "只测试，不提交预约"，点 **开始执行**，确认日志中能查到目标座位
6. 测试成功后取消勾选 dry-run，点状态栏的 **启用自动**，之后每天到点自动执行

> 步骤 3-6 全部在网页中完成，不需要碰 `config.yaml`。

## 日常使用

启动服务后什么都不用做，调度器每天在配置的时间自动提交预约。

- 查看状态：打开 `http://127.0.0.1:8765`，状态栏显示下次执行时间和上次结果
- 修改参数：网页中直接改 → 点 **保存计划** → 下次自动执行即按新参数
- 重新获取 cookie：cookie 过期后，浏览器重新登录，点 **获取cookie** 按钮刷新
- 临时手动执行：修改参数后点 **开始执行**（不影响自动调度）
- 暂停自动：点 **禁用自动**
- 执行日志：网页右侧实时显示，同时持久化到 `booking.log`

## 开机自启（可选）

右键以管理员身份运行 `setup_autostart.bat`，之后开机自动启动服务（后台静默运行）。

运行 `remove_autostart.bat` 即可移除。

## 网页控制台参数

| 参数 | 类型 | 说明 |
|---|---|---|
| 楼层 | 下拉框 | 六楼杭韵数阁 / 四楼宋韵云图 / 二楼东格物E堂 / 二楼数智渊阁 / 十二楼芯灵驿站 / 二楼西比特庭园 |
| 座位号 | 数字输入 | 目标座位编号 |
| 开始时间 | 下拉框 | 00:00 ~ 23:00 |
| 时长 | 下拉框 | 1 ~ 14 小时 |
| 执行时间 | 文本 | 定时提交时间，如 `20:00:00`；清空 = 立即提交 |
| 重试次数 | 数字 | 到点后最多重试次数（仅对"超出可预约座位时间范围"错误重试） |
| 重试间隔 | 数字 | 每次重试等待秒数 |
| 预约日期 | 单选 | 今天 / 明天 / 后天（仅本次执行生效，不保存） |
| 只测试 | 复选框 | 勾选后只查询不提交 |

## 状态栏按钮

| 按钮 | 作用 |
|---|---|
| **获取cookie** | 手动触发从浏览器读取登录态，cookie 过期后重新登录再点此按钮刷新 |
| **启用自动** | 开启每日定时自动预约 |
| **禁用自动** | 暂停自动预约 |
